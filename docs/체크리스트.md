#### 테스트 라이브러리 개발

- 우아한 테크 코스 과제를 하며 TDD 방식으로 미션을 진행해봄.
- Jest 라이브러리를 사용하며 JavaScript 테스트 라이브러리 동작 과정에 대해 관심이 생김.
- 오픈 미션을 계기로 mocking과 spy 등의 기능을 포함한 기능을 구현해보고자함.
- 또한 npm에 직접 라이브러리를 배포하는 경험을 쌓고 싶음.
